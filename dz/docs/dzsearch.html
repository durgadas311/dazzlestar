DazzleStar Searching

Searching of the program image may be done using a string (text) or a list of bytes entered in hexadecimal and separated by spaces. The radio buttons "Hex" and "Text" select which will be used. The radio buttons "Beginning" and "Current" select whether the search starts at the beginning of the program or the current (cursor) location. "Wrap" may be set to allow the search to wrap back to the beginning when reaching the end of the program.
